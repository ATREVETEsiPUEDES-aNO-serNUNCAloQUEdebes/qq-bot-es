1. para wiki Instale los complementos necesarios para la busqueda. [$:/plugins/EvidentlyCube/ExtraOperators](https://evidentlycube.github.io/TW5-PluginShowcase/#Extra%20Operators)，Puede arrastrar e instalar directamente desde este sitio web. Para obtener detalles sobre el uso de complementos, consulte [plugins/tw-qa/src/filter.ts](./plugins/tw-qa/src/filter.ts)
1. Crea un titulo como `$:/config/Server/AllowAllExternalFilters` Y el contenido es `yes` (Esto generalmente no debe hacerse para servidores publicos. Pero nuestrowikiSe ejecuta en la intranet y solo expone metodos de acceso limitado al exterior a traves del robot, por lo que puede ser asi）
